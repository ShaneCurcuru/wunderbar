Wunderbar Wiki

Note: demo code requires Ruby 1.9.3 or later. Wunderbar itself works with Ruby 1.8.7 or later.

Prereqs:

  • gem install wunderbar rdiscount ruby2js
  • apt-get install git
  • jQuery
  • Showdown

Create a data directory (one per wiki):

  • Must be writeable by your web server
  • Should not be in the Document Root

Configure git:

  • Follow Step 1.
  • Note that this needs to be done for the user that the webserver runs the script under. If you are not running suexec, this is typically www-data on Unix or www on Mac OS/X.

Configure Apache:

    Options +ExecCGI +MultiViews
    MultiViewsMatch Any
    AddHandler cgi-script .cgi

Optionally configure Authentication for multiple users.

Install scripts:

  • Copy showdown.js into your document root

Install wiki (one per wiki):

    ruby wiki.rb --install=/var/www/wikiname.cgi --WIKIDATA="/full/path/to/data/directory"

Optionally tailor the setting of the $EMAIL variable in the script. The current default logic is based on a Unix like operating system (such as Linux, Mac OS/X) with users having email addresses on this machine. Replace as needed.

Access your wiki:

    http://localhost/wikiname/
